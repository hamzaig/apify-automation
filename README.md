# LinkedIn Email Extractor

A modern Next.js application for extracting emails from LinkedIn profiles using Apify.

## Features

- Beautiful, modern UI with drag-and-drop file upload
- Secure token input with show/hide functionality
- Real-time processing status
- Detailed results display with success/failure statistics
- Download processed Excel files with extracted emails

## Getting Started

### Installation

Install the required dependencies using npm or yarn.

### Development

Run the development server to start the application locally.

### Production

Build the application for production deployment.

## Usage

1. Upload an XLSX file containing LinkedIn profile URLs
2. Enter your Apify API token
3. Click "Process File" to start extraction
4. Download the processed file with extracted emails

## Requirements

- Node.js 18+ 
- Apify API token
- XLSX file with LinkedIn profile URLs in a column containing "linkedin" in the header
